# Weather Dashboard

Projeto  dashboard climático utilizando HTML, CSS e JavaScript.

## Funcionalidades

- Pesquisa de cidades
- Consumo de API
- Temperatura em tempo real
- Umidade
- Velocidade do vento
- Favoritos
- Responsividade
- Alteração dinâmica de fundo
- LocalStorage

## Tecnologias

- HTML5
- CSS3
- JavaScript
- OpenWeather API
